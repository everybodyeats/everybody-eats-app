# everybody-eats-app

* First, clone the repo, download/install node from nodejs website, and run “npm install”
* Download the Expo app to your phone (Expo will let you see the app on your phone as you develop it)
* To run the project, type “npm start” in terminal, and then scan the qr code with your phone’s camera to run it in Expo
    * If you run into permission errors, use “sudo npm start” in front of the command
* When developing, everyone should be using their own branch
